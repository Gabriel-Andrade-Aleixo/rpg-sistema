# RPG Sheet Mobile

App mobile Flutter para criacao e gerenciamento de fichas.

Estrutura atual:

- `rpg_sheet_app`: mobile Flutter
- `../rpg_backend`: backend Node.js
- `../rpg_web`: web Next.js

Para rodar o mobile:

```bash
flutter run --dart-define-from-file=env.local.json
```
